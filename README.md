# proyecto-ionic
proyecto semestral de ionic

# prerequisitos:
tener instalado NodeJS

# pasos para que funcione:
* clonar git.
* ir a la carpeta del proyecto: cd registrapp.
* ejecutar npm install.
* ejecutar ionic: ionic serve
